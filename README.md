# 🌐 Domínio à Venda

Página de apresentação para divulgação de domínio à venda. Design minimalista com fundo em gradiente escuro e contato direto por e-mail.

## ✨ Preview

> Página centralizada com nome do domínio em destaque e e-mail de contato.

## 🚀 Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — gradiente escuro, glassmorphism com `backdrop-filter`, `text-shadow` com brilho roxo
- Sem frameworks ou dependências externas

## 📁 Estrutura

```
dominio-site1/
├── index.html   # Página principal
└── CNAME        # Configuração de domínio personalizado (GitHub Pages)
```

## 🖥️ Como rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/IgorASB/dominio-site1.git
   ```
2. Abra o arquivo `index.html` diretamente no navegador — sem necessidade de servidor.

## 🌍 Deploy

Hospedado via **GitHub Pages** com domínio personalizado configurado no arquivo `CNAME`.
